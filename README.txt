============================================================
  사내 리조트 잔여객실 통합 대시보드 시스템
  작업 현황 및 실행 방법
============================================================

■ 프로젝트 구성 현황
  ✅ 리솜 크롤러     : resom_crawler\resom_crawler.py
  ✅ 한화 크롤러     : hanhwa_crawler\hanhwa_crawler.py
  ✅ 소노 크롤러     : sono_crawler\sono_crawler.py
  ✅ 롯데 크롤러     : lotte_crawler\lotte_crawler.py
  ✅ 통합 변환기     : convert_to_html.py
  ✅ 배치 실행 파일  : 정찰실행.bat
  ✅ 공통 환경 설정  : .env (D:\휴양소\.env 위치)

============================================================
■ 로컬 실행 방법 (전체 일괄 실행)
============================================================

  1. D:\휴양소 폴더의 '정찰실행.bat' 파일을 더블클릭합니다.
  2. 배치 스크립트가 4개 브랜드(리솜 -> 소노 -> 롯데 -> 한화) 크롤러를 순차 실행합니다.
  3. 완료 후 자동으로 통합 HTML 변환이 진행됩니다.
  4. 생성된 'resort_availability.html' 파일을 브라우저로 열어 확인합니다.

============================================================
■ 로컬 실행 방법 (개별 실행)
============================================================

  ※ 각 크롤러 폴더로 이동한 후 개별 실행이 가능합니다.
  
  [리솜]
    cd /d D:\휴양소\resom_crawler
    python resom_crawler.py

  [소노]
    cd /d D:\휴양소\sono_crawler
    python sono_crawler.py

  [롯데]
    cd /d D:\휴양소\lotte_crawler
    python lotte_crawler.py

  [한화]
    cd /d D:\휴양소\hanhwa_crawler
    python hanhwa_crawler.py

  [통합 HTML 생성]
    cd /d D:\휴양소
    python convert_to_html.py

============================================================
■ 폴더 구조
============================================================

  D:\휴양소\
  ├── .env                    ← 전체 계정정보 공통 관리
  ├── resort_availability.html ← 최종 결과물 대시보드
  ├── convert_to_html.py      ← 통합 HTML 변환기
  ├── 정찰실행.bat             ← 일괄 실행용 배치 파일
  ├── resom_crawler\          ← 리솜 크롤러 폴더
  ├── hanhwa_crawler\         ← 한화 크롤러 폴더
  ├── sono_crawler\           ← 소노 크롤러 폴더
  ├── lotte_crawler\          ← 롯데 크롤러 폴더
  ├── 요금표\                 ← 분석용 요금표 PDF/엑셀 폴더
  └── 고도화_계획\            ← 구조 분석 및 고도화 보고서 폴더
